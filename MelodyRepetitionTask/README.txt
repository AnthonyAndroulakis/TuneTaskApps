#Dependencies:
Windows 8 OS
python 3
python 3 modules:
   1) pillow
   2) pyautogui
   3) pygame
   4) multiprocessing
   5) soundcard
ffmpeg

#Before you run MelodyRepetitionTask.py:
----I) put ffmpeg.exe in your PATH
        1) search for "Edit the system environment variables"
        2) under the "Advanced" tab, click "Environmental Variables..."
        3) under "User variables for..." (upper half), find the row with variable "Path" and click on it
        4) press the edit button (for the top half)
        5) click "New" and type in the path for ffmpeg (ex: C:\Program Files\ffmpeg\bin)
----II) install all necessary python3 modules

#Running MelodyRepetitionTask.py
 1) open cmd
 2) cd into the MelodyRepetitionTask directory
 3) double-click MelodyRepetitionTask.py
