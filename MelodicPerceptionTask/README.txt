DOCUMENTATION FOR MELODIC PERCEPTION TASK APP. CODE CREATED BY RYAN JOSEPH AND ANTHONY ANDROULAKIS.

#Dependencies:
Windows 8 OS
python 3
python 3 modules:
   1) pillow
   2) pyautogui
   3) pygame
   4) multiprocessing
   5) soundcard
a green button and a red button (linked to "1" and "2" keys, respectively)

#Before you run MelodyRepetitionTask.py:
----I) install all necessary python3 modules
----II) plug in green and red buttons & ensure that the green button is linked to the "1" key and the red button is linked to the "2" key

#Running MelodicPerceptionTask.py
 1) open cmd
 2) cd into the MelodicPerceptionTask directory
 3) double-sclick MelodicPerceptionTask.py
